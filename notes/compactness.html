Compactness

Topic: Topology of R

Motivation

Many powerful results in analysis — the extreme value theorem, uniform continuity on closed intervals, convergence of subsequences — all rely on one property: compactness. It is the precise way to say a set is "finite-like" even when it contains infinitely many points. Compact sets behave well: sequences can't escape them, continuous functions are bounded on them, and open covers can always be trimmed down to finitely many pieces.

Compactness is abstract because it quantifies over all possible open covers, not just specific ones. The goal of this note is to make that abstraction visual and concrete.

Definitions

Definition 1 — Open Covering

Let \((X, d)\) be a metric space and \(A \subset X\). An open covering (or open cover) of \(A\) is a collection of open sets \(\{U_\alpha : \alpha \in I\}\) such that

$$A \subset \bigcup_{\alpha \in I} U_\alpha.$$

If the index set \(I\) is finite, we call it a finite open covering and write \(\{U_{\alpha_1}, U_{\alpha_2}, \ldots, U_{\alpha_K}\}\).

Intuition

Picture the set \(A\) as a shape drawn on the floor. An open cover is a collection of overlapping spotlights (open sets) shining down, positioned so that every single point of \(A\) is illuminated by at least one spotlight. You're allowed to use as many spotlights as you want — even infinitely many. A "finite" cover just means you managed to do it with a limited number.

Definition 2 — Compactness

A subset \(A \subset X\) is called compact if every open covering of \(A\) has a finite sub-covering.

More precisely: for every open covering \(\{U_\alpha : \alpha \in I\}\) of \(A\), there exist finitely many indices \(\alpha_1, \ldots, \alpha_K \in I\) such that

$$A \subset U_{\alpha_1} \cup U_{\alpha_2} \cup \cdots \cup U_{\alpha_K}.$$

Intuition

No matter how someone tries to cover \(A\) with open sets — even with an absurdly wasteful infinite pile of them — you can always throw away all but finitely many and still have \(A\) fully covered. The key word is every: this must work for any cover anyone could dream up.

Why does \([0, 1]\) pass but \((0, 1)\) fail? For the open interval, an adversary can build a cover where each set barely extends inward from 0 — like \((1/n, 2)\) for \(n = 1, 2, 3, \ldots\). No finite selection of these covers all of \((0, 1)\) because points near 0 keep needing the next smaller interval. The closed interval \([0, 1]\) blocks this trick because 0 is in the set, and any open set covering 0 must swallow a neighborhood around it, covering infinitely many of the "problematic" points at once.

Definition 3 — Sequential Compactness

A subset \(A\) of a metric space \((X, d)\) is called sequentially compact if every sequence in \(A\) has a convergent subsequence whose limit lies in \(A\).

Intuition

Imagine dropping infinitely many marbles inside a room shaped like \(A\). Sequential compactness says the marbles must pile up somewhere inside the room — they can't all spread out evenly forever, and they can't all drift toward a wall that isn't part of the room. If \(A = [0, 1]\), no matter how you scatter the marbles, some subsequence of them will cluster at a point that is still inside \([0, 1]\). If \(A = (0, 1)\), you could place marbles at \(1/n\) — they cluster at 0, but 0 isn't in the room, so the set fails.

Theorems

Theorem 1 — Compact iff Sequentially Compact

In any metric space, a set is compact if and only if it is sequentially compact.

The proof is long and therefore omitted (see e.g. Rudin, Theorem 3.6).

Theorem 2 — Closed Subset of a Compact Set is Compact

Let \((X, d)\) be any metric space, \(K \subset X\) compact, and \(F \subset K\) closed. Then \(F\) is compact.

Proof

Goal: given any open cover of \(F\), extract a finite subcover.

  • Let \(\{U_\alpha\}\) be an open cover of \(F\).
    — Start with an arbitrary open cover of the closed subset \(F\).
  • Since \(F\) is closed, \(F^c = X \setminus F\) is open.
    — We'll use the complement to extend the cover to all of \(K\).
  • The collection \(\{U_\alpha\} \cup \{F^c\}\) is an open cover of \(K\): every point of \(K\) is either in \(F\) (covered by some \(U_\alpha\)) or in \(F^c\).
    — By adding one more open set, we turn a cover of \(F\) into a cover of the compact set \(K\).
  • Since \(K\) is compact, extract a finite subcover: \(K \subset U_{\alpha_1} \cup \cdots \cup U_{\alpha_n} \cup F^c\).
    — Compactness of \(K\) gives us finitely many sets.
  • Then \(F \subset U_{\alpha_1} \cup \cdots \cup U_{\alpha_n}\) (since \(F \cap F^c = \emptyset\), the \(F^c\) term contributes nothing to covering \(F\)).
    — Drop \(F^c\) and we have a finite subcover of \(F\). Done.
Theorem 3 — Compact Implies Closed and Bounded

In any metric space, every compact set is closed and bounded.

Proof (bounded)

Goal: show a compact set \(K\) fits inside some ball of finite radius.

  • Fix any point \(p \in X\). The open balls \(\{B_n(p)\}_{n=1}^{\infty}\) form an open cover of \(K\) (indeed, of all of \(X\)).
    — Every point in \(X\) is at finite distance from \(p\), so it lies in some \(B_n(p)\).
  • By compactness, extract a finite subcover: \(K \subset B_{n_1}(p) \cup \cdots \cup B_{n_k}(p)\).
    — Finitely many concentric balls suffice.
  • Let \(N = \max(n_1, \ldots, n_k)\). Then \(K \subset B_N(p)\), so \(K\) is bounded.
    — The largest ball contains all the others, giving a single finite bound.
Proof (closed)

Goal: show \(K\) contains all its limit points — equivalently, \(K^c\) is open.

  • Let \(q \notin K\). For each \(x \in K\), let \(r_x = d(q, x)/2 > 0\). The balls \(\{B_{r_x}(x)\}_{x \in K}\) cover \(K\).
    — We build an open cover of \(K\) using balls centered at points of \(K\), each half the distance to \(q\).
  • By compactness, extract a finite subcover: \(K \subset B_{r_{x_1}}(x_1) \cup \cdots \cup B_{r_{x_n}}(x_n)\).
    — Finitely many balls around points \(x_1, \ldots, x_n\) cover \(K\).
  • Set \(r = \min(r_{x_1}, \ldots, r_{x_n}) > 0\). Then \(B_r(q) \cap K = \emptyset\).
    — Any point within distance \(r\) of \(q\) is too far from every \(x_i\) to be in the corresponding ball. So \(q\) has an open neighborhood disjoint from \(K\).
  • Since every \(q \notin K\) has an open ball around it missing \(K\), the complement \(K^c\) is open, so \(K\) is closed.
    — This is exactly the definition of a closed set.
Theorem 4 — The Heine-Borel Theorem

In \(\mathbb{R}^n\) with the Euclidean metric, a set is compact if and only if it is closed and bounded.

Proof sketch

Goal: the hard direction is "closed and bounded ⇒ compact." Theorem 3 gives the other direction.

  • (⇒) Already proved: Theorem 3 shows compact implies closed and bounded in any metric space.
    — This direction doesn't need the Euclidean structure at all.
  • (⇐) Suppose \(A \subset \mathbb{R}^n\) is closed and bounded. Since \(A\) is bounded, it sits inside a closed box \([-M, M]^n\).
    — We reduce to showing the box is compact, then use Theorem 2 (closed subset of compact is compact).
  • The key step is proving \([-M, M]^n\) is compact. This is done by contradiction: assume an open cover has no finite subcover, bisect the box repeatedly, and use the nested intervals theorem (Bolzano-Weierstrass) to find a point that must be covered yet leads to a contradiction.
    — This bisection argument is specific to \(\mathbb{R}^n\) and does not work in general metric spaces.
  • Since \(A\) is a closed subset of the compact set \([-M, M]^n\), Theorem 2 gives that \(A\) is compact.
    — The two prior theorems combine to finish the proof.

Examples

Example 1 — Singletons are compact

Any singleton \(\{x\}\) is compact: given any open cover, at least one set \(U_\alpha\) contains \(x\), so \(\{U_\alpha\}\) alone is a finite subcover.

Example 2 — Finite unions of compact sets are compact

If \(K_1, \ldots, K_N\) are compact, then \(\bigcup_{j=1}^{N} K_j\) is compact. Given any open cover of the union, it covers each \(K_j\), so extract a finite subcover for each. The union of these \(N\) finite subcovers is still finite and covers the whole union.

Example 3 — Closed but not compact

\(\mathbb{R}\) itself is closed (its complement \(\emptyset\) is open) but not compact. The cover \(\{(-n, n) : n \in \mathbb{N}\}\) has no finite subcover, because any finite selection is bounded by \(\max(n_1, \ldots, n_k)\), missing points beyond that bound.

Example 4 — Bounded but not compact

\((0, 1)\) is bounded but not compact (it is not closed). The cover \(\{(1/n, 1) : n \geq 2\}\) covers \((0, 1)\) but no finite subcollection does: for any finite selection with largest index \(N\), points in \((0, 1/N)\) are missed.

Example 5 — Closed and bounded but not compact (general metric space)

Let \(X = \mathbb{N}\) with the discrete metric (\(d(m, n) = 1\) if \(m \neq n\), \(d(m, m) = 0\)). Then \(X\) is closed (every set is closed in the discrete metric) and bounded (every pair of points has distance at most 1). But \(X\) is not compact: the cover \(\{\{n\} : n \in \mathbb{N}\}\) consists of open singletons with no finite subcover. This shows Heine-Borel fails outside \(\mathbb{R}^n\).

Exercises

Exercise 1

Prove directly from the definition that any finite set \(\{x_1, x_2, \ldots, x_n\}\) in a metric space is compact.

Show solution

Let \(\{U_\alpha\}\) be any open cover. For each \(x_i\), pick one \(U_{\alpha_i}\) containing \(x_i\). Then \(\{U_{\alpha_1}, \ldots, U_{\alpha_n}\}\) is a finite subcover, since every point of the set is in some \(U_{\alpha_i}\).

Exercise 2

Let \(K_1, \ldots, K_N\) be compact subsets of a metric space. Prove that \(\bigcup_{j=1}^{N} K_j\) is compact.

Show solution

Let \(\{U_\alpha\}\) be an open cover of \(\bigcup K_j\). It is also an open cover of each \(K_j\). By compactness of \(K_j\), extract a finite subcover \(\mathcal{F}_j \subset \{U_\alpha\}\). Then \(\mathcal{F}_1 \cup \cdots \cup \mathcal{F}_N\) is a finite subcover of the union (a finite union of finite sets is finite).

Exercise 3

Give an example of a subset of a metric space that is both closed and bounded but not compact. (Hint: consider the discrete metric on an infinite set.)

Show solution

Take \(X = \mathbb{Z}\) with the discrete metric. The entire set \(\mathbb{Z}\) is closed (every set is closed in the discrete topology) and bounded (all pairwise distances are \(\leq 1\)). But the open cover \(\{\{n\} : n \in \mathbb{Z}\}\) (each singleton is open in the discrete metric) has no finite subcover, so \(\mathbb{Z}\) is not compact. This works because Heine-Borel applies only to \(\mathbb{R}^n\) with the Euclidean metric.
